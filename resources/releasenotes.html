📣 MyTradingTracker Release Notes - Version 1.0.4

We are excited to present the latest update for MyTradingTracker!
This version comes packed with new features, improvements, and bug fixes to enhance your trading experience. Here are the highlights:

New Features

  • 🌟Dark Mode is now available!
  • ⭐️Pagination on the Trades page

Bug Fixes

  • 🐛None

Improvements

  • 🚀API improvements to increase security and limit user data exposure
  • 💡Filter input for ticker name will now also serve as a search bar against all of your trades logged in the app
  • ⚙️Animation added to Home page for users who have not added any trades yet
  • ⭐️User can now enter any ticker they want when adding/editing a trade if it does not show up in search dropdown
  • 🔧Styling improvements to email templates

What's Coming Next

  • 🧑‍💻Ability to track your trading account value based on trades entered into MyTradingTracker
  • 🔜Journaling Page for you to reflect on your strategy or track your trading rules
  • 🔄Caching to improve speed
  • 🔧Ability to see which filters are curently applied
  • ⚙️Backend enhancements to improve speed and decrease sending of unecessarry data
