📣 MyTradingTracker Release Notes - Version 1.1.1

We are excited to present the latest update for MyTradingTracker!
This version comes packed with new features, improvements, and bug fixes to enhance your trading experience. Here are the highlights:

New Features

  • ⭐TRADE-139 - Profile Pictures - Users can now Upload and Update their Profile Icon to any image of their choosing.
  • ⭐TRADE-157 - 2FA - Two-Factor Authentication via email can now be enbaled in User Preferences.
  • ⭐TRADE-264 - Leaderboard - User Leaderboard that compares various statistic values over different periods of time for Public Profiles only. Users can make their profile public or private in User Preferences.
  • ⭐TRADE-496 - CSV Bulk Trade Updates - You can now update specific values on existing trades in bulk via CSV.

Bug Fixes

  • 🐛TRADE-541 - Trade Date formats are now stored and displayed uniformly, accomodating multiple date formats as input and invalid date formats will be blocked.
  • 🐛TRADE-558 - Resolved any combination of Updating a Trade(s) (Bulk, Single, CSV) where you could update Options/Shares and inappropriately fail on Expiry & Strike Validation.

Improvements

  • 🔧TRADE-539 - Updates made to support Heroku's changes for their Redis component.
  • 🚀TRADE-543 - When Importing Trades via CSV, users can now specify if these imports should affect their Account Value data or not via a checkbox.
  • 🔧TRADE-544 - Preferred Currency dropdown is now a searchbar, making it easier to find your desired Currency Code.
  • 🚀TRADE-545 - Added an additional check if a user is Updating a Trade(s) that pnl, # of units and % win/loss are all present for more accurate calculations.
  • 🔧TRADE-629 - Emergency Database Migration from ClearDB provider to JawsDB, this is what unfortauntely led to ~3 days of downtime for MyTradingTracker. You may need to manually update your Account Value, as it was likely overwritten to 0 over the downtime period and week prior. Historical data prior to this date are still preserved and only Account Value was affected.

What's Coming Next

  • 🧑‍💻Addition of commision/fees in each trade entry for more accurracy on accout value calculation, and opportunity for insight into long term spending on fees.
  • 🧑‍💻Additional classification for each trade entry with PUT/CALL for options and LONG/SHORT for equity trades.
  • 🧑‍💻Addition of Front End automated testing.
  • 🧑‍💻Addition of Trade reccomendations based on your trade history, providing insight into your best trades and trading strategy.
  • 🧑‍💻Increased security for Reset and 2FA codes.
  • 🧑‍💻2FA enchanements, adding Phone Number and other 2FA options.
