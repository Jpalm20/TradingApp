📣 MyTradingTracker Release Notes - Version 1.0.5

We are excited to present the latest update for MyTradingTracker!
This version comes packed with new features, improvements, and bug fixes to enhance your trading experience. Here are the highlights:

New Features

  • 🌟Account Balance Tracking based on trades entered into MyTradingTracker is now available!

Bug Fixes

  • 🐛Fixed a bug that was keeping users from fully Deleting their account... maybe should have kept this in, lol!

Improvements

  • 🚀API improvements when getting Trade Statistics in order to increase speed drasitcally

  • ⚙️Risk/Reward input when adding or updating a trade improved, making it more user friendly
  • ⭐️Improved error forwarding from the Backend
