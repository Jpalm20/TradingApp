📣 MyTradingTracker Release Notes - Version 1.0.7

We are excited to present the latest update for MyTradingTracker!
This version comes packed with new features, improvements, and bug fixes to enhance your trading experience. Here are the highlights:

New Features

  • 🌟Landing Page - A new look to the start of your journey with MyTradingTracker!
  • ⭐️Email Opt-In/Out - You can now enable or disable email notification in the settings of the User Profile Page.
  • 📊Redis Caching - Enjoy decreased data access latency with caching, better put... faster response time when using MyTradingTracker!
  • 📝Logging - Added server side logging to aid in troubleshooting and security by saving all application events.

Bug Fixes

  • 🐛On desktop, options contract expiry dates can now be selected in the future, this was inappropriately blocked previously.
  • 🕷️When trying to load Journal Page during December there was an error due to incorrect date handling for overflow into next year, now resolved.
  • 🐞Issue when trying to submit feedback in the form of Bug Report or Feature Request due to invalid string concatentation, now resolved.

Improvements

  • 🔧Account Value Tracking enhancement - Account Value chart is now anchored to $0 for a better perspective and outlook on account growth.
  • 🚀Journal Page enhancements - Rich Text Support, auto sizing for the entry box and improved experience when cancelling an entry before saving!

What's Coming Next

  • 🧑‍💻Wrapping up the build out of our Frontend and Backend testing for a more robust and secure application, all tests will need to pass before releasing a new build to Production.

🚀 Check out the latest version now and take your trading journey to new heights! Happy trading! 📈💰

Stay tuned for more exciting updates in the future! 💪
