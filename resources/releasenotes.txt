Version 1.0.1 - 05-18-2023


Features:

- Trade Summary Page Improvements:
    - Ability to select and delete multiple trades at once
    - 'Select All' button which will select all trades with the push of a button
    - 'Clear' button which will clear your current selection of trades

- Ticker Name Search Bar when updating or adding a trades
    - Autocomplete search bar to help you find the exact ticker you need

- Strong Password Suggestion
    - When signing up for a new account, you are now prompted to generate a strong password in one button press
    - Promp will go away if you start typing your own password


Bug Fixes:

- None
